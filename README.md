## GitHub Pages setup
1. In the repo, go to Settings -> Pages.
2. Under Build and deployment, set Source to Deploy from a branch.
3. Select Branch: main (or master) and Folder: /(root).
4. Save. Then open: /login.html

## Notes
- This is client-side only (localStorage). It is not secure authentication.
- Intended for training / SOP gating.
